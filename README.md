# numsjs
This is a JS module for calculation and mathematics

## Installation
npm install numsjs

## Usage
var { add, sub, mul, div } = require('numsjs')<br />
let add_2 = add(1, 2) //get 3 <br />
let mul_3 = mul(1, 2, 3) // get 6 <br />
let div_0 = div(3, 0) // get INFINITY <br />
